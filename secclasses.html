Discord Issue Severity Classification

Classes

  • Class 0: 😐 (Neutral face) - Minimal impact issues, occasional minor server lag affecting a few users.
    • Sub-tier 1: 🙂 (Slightly smiling face) - Rare and brief lag, hardly noticeable to most users.
    • Sub-tier 2: 😶 (Neutral face) - Occasional short periods of lag, minimal impact on user experience.
    • Sub-tier 3: 😕 (Slightly frowning face) - Intermittent and slightly noticeable lag, occasional inconvenience for a few users.
    • Sub-tier 4: 😟 (Worried face) - More frequent lag but still infrequent enough to cause minor frustration.
    • Sub-tier 5: 😩 (Tired face) - Frequent but brief lag, causing mild annoyance for some users.
  • Class 1: 😕 (Slightly frowning face) - Moderate impact issues, noticeable server lag affecting a larger portion of users.
    • Sub-tier 1: 😟 (Worried face) - Intermittent lag, occasional inconvenience for some users.
    • Sub-tier 2: 😩 (Tired face) - Frequent lag, causing frustration for many users, impacting usability.
    • Sub-tier 3: 😣 (Persevering face) - Consistent and significant lag, hindering normal interactions and functionalities.
    • Sub-tier 4: 😤 (Face with steam from nose) - Pervasive lag, severe impact on server activities, urgent attention needed.
    • Sub-tier 5: 😡 (Pouting face) - Extreme and constant lag, rendering server nearly unusable, requires immediate resolution.
  • Class 2: 🚧 (Construction sign or wrench) - Scheduled maintenance resulting in temporary server downtime.
  • Class 3: 🔒 (Locked padlock) - Security concerns.
    • Sub-tier 1: 🔐 (Locked with pen) - Suspicious activity, requires monitoring and investigation.
    • Sub-tier 2: 🛡️ (Shield) - Attempted unauthorized access but unsuccessful, heightened security measures advised.
    • Sub-tier 3: ⚠️🔒 (Warning and locked padlock) - Confirmed unauthorized access to restricted areas, urgent action required.
  • Class 4: ⚠️ (Warning sign) - Severe disruptions.
    • Sub-tier 1: ⚠️⚠️ (Double exclamation mark) - Indicates disruptive behavior, immediate moderation needed.
    • Sub-tier 2: ❗ (Exclamation mark) - Coordinated disturbance, multiple users causing substantial disruption.
    • Sub-tier 3: ❗❗ (Double exclamation marks) - Severe server disruption, significant impact on user experience, requires swift action.
  • Class 5: 🚨 (Emergency siren) - Critical emergencies.
    • Sub-tier 1: 🚨🚨 (Double emergency sirens) - Critical server instability, most functionalities compromised, critical actions needed.
    • Sub-tier 2: 🆘 (SOS button) - Server on the verge of collapse, complete outage imminent, urgent intervention necessary.
    • Sub-tier 3: 🚨🆘 (Emergency siren and SOS button) - Server in imminent danger of complete failure, unprecedented emergency, immediate expert intervention essential.
  • Danger Class: 💥 (Explosion) - Catastrophic scenarios, surpassing Class 5 emergencies, extreme and urgent situations.
